# プライバシーポリシー

アプリ名：アルキスト / 開発者：YUJI KANEKO

## 1. はじめに

本ポリシーは、「アルキスト」（以下「本アプリ」）におけるユーザー情報の取り扱いについて説明するものです。本アプリをご利用いただく際は、本ポリシーをご確認のうえ、ご同意いただいたものとみなします。

本アプリは散歩の継続を支援するためのツールです。表示される歩数・距離・天気情報・UV指数等は参考情報であり、医療行為や医学的助言を提供するものではありません。

## 2. 収集する情報

- **歩数データ：** iOS のヘルスケア（HealthKit）から読み取られ、端末内でのみ処理されます。書き込みは行いません。
- **位置情報：** 散歩中の経路の概略の把握、写真への位置情報付与、天気情報の取得に使用され、端末内に保存されます。
- **撮影した写真：** 本アプリ内のカメラで撮影した写真は、撮影時の位置情報とともに端末内に保存されます。
- **散歩記録：** 開始時刻・終了時刻・歩数・場所メモといった情報が端末内に保存されます。

## 3. データの取り扱い

- 上記すべての情報は、端末内のアプリストレージにのみ保存されます。
- 開発者（YUJI KANEKO）のサーバや第三者サービスにアップロードされることはありません（次項の天気情報取得を除く）。

## 4. 天気情報の取得

本アプリは、散歩開始時の気温・UV指数・天候を表示するため、外部の天気情報API「**Open-Meteo**」（<https://open-meteo.com>）に位置情報（緯度・経度）を送信します。送信される情報は緯度・経度のみで、個人を特定する情報は含まれません。

詳細は Open-Meteo の利用規約・プライバシーをご確認ください：<https://open-meteo.com/en/terms>

## 5. 広告について

本アプリは現時点で広告を表示しません。将来的に広告サービスを導入する場合は、本ポリシーを更新したうえで、アプリのアップデート等を通じてお知らせします。

## 6. データの保存と削除

- すべてのデータは端末内にのみ保存され、外部に送信されることはありません（前項の天気情報取得を除く）。
- 本アプリの設定画面の「データを全て削除」機能から、いつでもすべての記録データを削除できます。
- 本アプリを端末からアンインストールすると、すべてのデータが消去されます。
