# アルキスト (Arukist)

散歩の継続を支援する iOS アプリ。

## 機能

- 歩数の自動カウント（HealthKit 連携）
- 散歩中の写真記録
- 場所ベースの「マイスポット」育成
- 天気・UV指数の表示
- すべてのデータは端末内に保存（外部送信なし）
